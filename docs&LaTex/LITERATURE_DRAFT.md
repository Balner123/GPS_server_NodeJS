# Návrh seznamu literatury a zdrojů (Bibliografie)

Tento seznam obsahuje technickou dokumentaci, datasheety a oficiální manuály k technologiím použitým v projektu LOTR. Je připraven pro pozdější vložení do LaTeX souboru.

5.  **SHYMANSKYY, Volodymyr.** *TinyGSM: A small Arduino library for GSM modules* [Software]. GitHub, 2023. Dostupné z: https://github.com/vshymanskyy/TinyGSM
